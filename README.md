# Niche CRM MVP

**AI-Powered CRM for [VERTICAL TO BE CHOSEN]**

## Revenue Model
- **Pricing:** $99-$299/month per business
- **Target:** 100-500 customers
- **Goal:** $10K-$50K MRR within 12-18 months

## Core Features (MVP)
1. **Contact Management** - Store and organize customer data
2. **Lead Tracking** - Pipeline visualization and status updates
3. **AI Follow-Up Assistant** - Automated email/SMS reminders
4. **Appointment Scheduling** - Calendar integration
5. **Activity Timeline** - Full customer interaction history

## Tech Stack
- **Frontend:** React + TypeScript
- **Testing:** Vitest + React Testing Library
- **State:** Zustand (lightweight, fast)
- **Data Fetching:** TanStack Query
- **Validation:** Zod
- **AI:** OpenAI API (follow-ups, lead scoring)

## Development

```bash
npm install
npm run dev        # Start dev server
npm run test       # Run tests
npm run test:ui    # Open Vitest UI
npm run build      # Production build
```
